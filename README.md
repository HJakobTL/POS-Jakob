## Repository-Ordnung

| Ordner | Beschreibung |
| :--- | :--- |
| **`Einsteigerkurs`** | Aufgaben und Codebeispiele für den ersten Teil des Kurses (BlueJ). |
| **`Zweisteigerkurs`** | Aufgaben und Codebeispiele für den zweiten/fortgeschrittenen Teil des Kurses (IntelliJ). |
| **`PLF1_Loes`** | Lösungen oder Referenzcode für die Prüfungsleistung 1 (PLF1). |
| **`Test`** | Ordner für allgemeine Tests, kleine Code-Experimente oder temporäre Dateien. |

## Technologie

Dieses Projekt basiert hauptsächlich auf der Programmiersprache Java.
